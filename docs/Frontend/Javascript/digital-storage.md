# JavaScript 中数字的存储

首先提出如下问题：

- JS 中的小数运算是精确的吗？【不是】
- JS 中的整数运算是精确的吗？【不是】
- JS 中的整数运算是连续的吗？【不是】
- JS 中能表示的最大数字是多少？【最大连续整数 Number.MAX_SAFE_INTEGER】
- JS 中能表示的数字的有效位数是多少？【16~17】

:::warning
最大连续整数是指向下连续且 +1 仍然连续，并非字面意义“最大”！
:::
