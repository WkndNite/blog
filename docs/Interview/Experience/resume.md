# 简历准备和复盘

## 实习经历

- 工作背景：司内有适配于字节商业化能力的一站式广告解决方案，将不同的广告系统抽象为产品用于集中管理，解决了随着存量业务增长引起的效率收益稀释的问题。其中，我主要负责建设和完善广告算法相关的平台，也参与部门内部自研的元数据管理相关工作。此外，负责一些数据口径升级更新、用户体验优化等日常维护工作。
- 核心技术：Eden(Monorepo) + Garfish(微前端) + React + TypeScript + Zustand + Express。
- 主要业绩：

  - 前期快速熟悉存量业务，在现有看板的基础上，计算并添加新的质量指标，实现数据可视化，提高了内部开发人员对特征质量的管控。
  - 鉴于跨项目引用，重构现有代码完成局部抽离，实现了不可编辑状态的图组件的封装，提高了代码复用率并且保证了后期的可维护性。
  - 主导了特征淘汰部分的开发任务，实现了从零到一的模块开发，包括特征淘汰任务的发起、特征集的状态标记、原始特征的选取等功能。另外，实现了飞书接口的 BFF 开发，提供了任务流程中的相关人员的拉群、催办等接口。
  - 基于已有的代码和相似的功能，实现复用并完善了特征离线调研任务的管理，页面内嵌编辑器，支持模型代码的编辑与保存、代码版本的回溯以及配置调试等。
  - 参与了元数据的研发，考虑到新人不熟悉元数据查询语法以及复杂算子的使用，主导了 PE 并针对幻觉等现象提出后续优化方案，实现了通过 AI 将自然语言转换为 DSL。其中，简单算子准确度达到 79.16%，复杂算子准确度基本达到 82.5%。另外，在该部分功能添加了埋点和相应的看板，实现了 AI 生成率和回答采纳率的数据上报。
