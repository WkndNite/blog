# 腾讯音乐暑期实习初试

1. 自我介绍并讲一下为什么选择前端。
2. 看你的低代码平台项目用到了部署，你是怎么部署的？
3. 手动部署和自动部署有什么区别？
4. 如何让部署过程中和你的钩子交互？
5. 讲一讲对 JavaScript 的理解。
6. 有没有深入学习过网络？如何学习的网络？
7. 在前端你觉得哪里会用到网络知识？
8. 讲一讲跨域问题以及常用的解决办法。
9. 看你用过 Webpack/Vite，挑一个讲一讲背景和作用。
10. 工程化有没有了解过 AMD/CMD？
11. 讲一讲括号匹配问题怎么解决。
12. 作为大三的学生，你业余时间怎么安排？
13. 说一说你未来的路线和规划。
14. 「反问」聊到部署，在部门内部需要前端负责部署嘛？
15. 「反问」部门内部技术栈是什么？
16. 「反问」对学习路线和规划的个人建议。

> [!Important] 面试感受
> 腾讯音乐面试略微轻松，不过有些问题面试结束后感觉自己答得不够多，典型的嘴比脑子快。
>
> **所以面试的时候大家要预留一些思考时间再回答。**
>
> 整体来说能感受到有超出面试官预料，11：30 结束一面，15：25 收到复试通知。
